# Plan

## Phase 0 (2-3 weeks): foundation
- repo structure and build system (CMake + Vite)
- core data model and common utilities
- CI skeleton and formatting baseline

## Phase 1 (3 months): MVP
- market data collector (C)
- basic indicators and signal pipeline (C++)
- simple backtest runner
- minimal REST API and web dashboard

## Phase 2 (6 months): operational system
- risk engine with limits and VaR
- order manager with paper trading
- persistent storage and replay
- monitoring and alerting

## Phase 3 (12 months): product
- ML models and feature store
- multi-venue connectors
- compliance reporting and audit trails
- HA, scaling, and security hardening
